# 中文取名神器 🎯

一个智能的中文取名 H5 网站，帮助父母为孩子取一个美好的名字。

## ✨ 功能特性

- 🎨 **多种风格选择**：古典诗意、现代简约、文雅书香
- 👶 **性别适配**：支持男孩、女孩、中性名字
- 📅 **生辰八字**：可选择出生日期进行五行分析
- ⭐ **智能评分**：综合音韵、字义、五行等多维度评分
- 💝 **收藏功能**：保存喜欢的名字，方便对比选择
- 📖 **名字详情**：查看字义解析、文化出处、五行属性
- 🧪 **名字测试**：测试已有名字的综合评分
- 📚 **取名知识**：学习取名技巧和注意事项

## 🛠️ 技术栈

- **前端框架**：Vue 3 + Vite
- **UI 组件库**：Vant 4
- **样式方案**：TailwindCSS
- **状态管理**：Pinia
- **路由管理**：Vue Router 4

## 📱 页面路由

- `/` - 首页（取名表单）
- `/generate` - 生成结果页
- `/detail/:name` - 名字详情页
- `/favorites` - 我的收藏
- `/test` - 名字测试
- `/knowledge` - 取名知识库

## 🎯 核心功能说明

### 1. 智能取名

根据用户输入的姓氏、性别、风格偏好等信息，智能生成 15 个候选名字，每个名字包含：
- 完整姓名和拼音
- 综合评分（0-100分）
- 字义解析
- 五行属性
- 音韵分析

### 2. 名字详情

点击任意名字可查看详细信息：
- 每个字的详细含义
- 五行属性分析
- 音韵和谐度评价
- 文化出处和典故

### 3. 收藏管理

- 支持收藏喜欢的名字
- 本地存储，刷新不丢失
- 可随时查看和管理收藏列表

### 4. 名字测试

输入已有的名字，系统会给出：
- 综合评分
- 音韵评分
- 字义评分
- 五行评分

## 🔧 配置说明

### Vite 配置

项目使用 Vite 作为构建工具，配置文件为 `vite.config.js`。
